# Pizza Place Sales Dataset (Pizza Sales Analysis)

This repository contains a cleaned/analysis-ready version of a pizza sales dataset and example features derived from it (e.g., `sales`, `months`, `week_days`, `hour`, `time_period`). The dataset can be used to explore product performance, time-based demand patterns, and pricing/revenue insights.

## Analysis done on the dataset
- **Revenue & demand:** top-selling pizzas, best categories/sizes, revenue drivers (price vs volume)
- **Time patterns:** peak hours, best days of week, monthly trends, morning/afternoon/evening performance
- **Order behavior:** number of orders per pizza/size, average order metrics
- **Product mix:** size preferences per pizza, category distribution, ingredient-based insights
